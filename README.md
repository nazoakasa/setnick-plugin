# setnick

Minecraft（Spigot/Bukkit）用プラグイン。

## 機能
- `/setnick <nickname>` コマンドで自分のニックネームを自由に変更できます。
- カラーコード（例: &a, &b など）も使用可能です。

## インストール方法
1. このリポジトリをビルドし、生成されたJARファイルを `plugins` フォルダに入れてください。
2. サーバーを再起動、または `/reload` してください。

## 使い方
- `/setnick <nickname>`
  - 例: `/setnick &aグリーンネーム`
